Desempenho e Custo dos Modelos MLLM

Objetivo: Avaliar a eficiência e o custo computacional médio dos diferentes modelos de Inteligência Artificial utilizados, mapeando o tempo de processamento e a quantidade de tokens gastos (Tabelas: modelo_mllm, inferencia, laudo).

/* Código SQL Executado no Servidor */

SELECT
    m.Nome_Modelo,
    COUNT(i.ID_Inferencia) AS Total_Inferencias,
    ROUND(AVG(i.Tempo_Processamento), 0) AS Tempo_Medio_MS,
    ROUND(AVG(l.Tokens_Consumidos), 0) AS Media_Tokens
FROM modelo_mllm m
JOIN inferencia i ON m.ID_Modelo = i.ID_Modelo
JOIN laudo l ON i.ID_Inferencia = l.ID_Inferencia
GROUP BY m.Nome_Modelo;

Carga Operacional por Equipamento

Objetivo: Analisar a utilização da infraestrutura hospitalar, exibindo o total de exames realizados e a soma do volume de dados em disco (MB) gerado por cada máquina (Tabelas: equipamento, exame, imagem_medica).

/* Código SQL Executado no Servidor */

SELECT
    eq.Nome_Equipamento,
    COUNT(DISTINCT ex.ID_Exame) AS Total_Exames,
    ROUND(SUM(im.Tamanho_Arquivo_MB), 2) AS Volume_Total_MB
FROM equipamento eq
JOIN exame ex ON eq.ID_Equipamento = ex.ID_Equipamento
JOIN imagem_medica im ON ex.ID_Exame = im.ID_Exame
GROUP BY eq.Nome_Equipamento;

Aceitação Clínica e Auditoria

Objetivo: Quantificar o nível de precisão da IA através das revisões médicas (concordâncias e discordâncias) em laudos gerados (Tabelas: revisao_laudo, medico, laudo).

/* Código SQL Executado no Servidor */

SELECT
    rl.Concordancia,
    COUNT(rl.ID_Revisao) AS Quantidade_Revisoes,
    ROUND(AVG(l.Tokens_Consumidos), 0) AS Media_Tokens_Gastos
FROM revisao_laudo rl
JOIN medico m ON rl.ID_Medico = m.ID_Medico
JOIN laudo l ON rl.ID_Laudo = l.ID_Laudo
GROUP BY rl.Concordancia
ORDER BY Quantidade_Revisoes DESC;
